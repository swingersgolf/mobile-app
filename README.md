# Swingers Mobile App

This repository contains the mobile application for Swingers.

## Tech Stack

- Expo (React Native)
- TypeScript
- Jest
- EAS
- GitHub Actions

## Prerequisites

- Node.js (v20.15.1)
- Expo CLI (SDK 51)

## Getting Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
